🏗️ Streaming Architecture Overview

  • Data Producer Producer
    Generates demand events every second, simulating IoT sensors or user interactions
  • Message Queue Queue
    Python Queue acts as Kafka topic - buffers messages between producers and consumers
  • Data Consumer Consumer
    Processes events in real-time, calculates pricing, maintains latest data buffer
  • REST API
    Flask endpoints provide real-time access to processed streaming data
